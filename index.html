    • Summary: Latitude vs Weather

    The purpose of the project was to visualise the impact of the latitude on four different weather features: Max Temperature, Cloudiness, Wind Speed and Humidity. More than 500 cities across the world were randomly selected through a Python library representing a sizable sample.

    Once the sample of cities was consolidated showing their coordinates, the four weather features were retrived for each city using the OpenWeatherMap API. The visulizations were reproduced using Matplotlib showing scatter plots of the whole sample comparing the latitude with each weather feature.
